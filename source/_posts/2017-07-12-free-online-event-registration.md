以前在 NPO(Nonprofit Organization，非營利機構) 時，常舉辦許多活動，當時是自行開發活動報名系統，但是以需求面而言，假如後續資料不再處理利用，或是為一次性活動，則可以考慮採用一些免費的的工具完成，方便參加者直接線上填寫資料、繳費報名，甚至使用手機 App 的電子票券。對主辦方來說，這些資料可以直接轉存成 Excel 等格式，方便資料的蒐集、存檔與再利用。<!-- more -->

以下介紹幾種不同的線上服務，分為表單類型的服務以及報名系統的服務，可依據不同的活動需求來選擇。

- [KKTIX](https://kktix.com/)
{% note success %}
免費活動限制與提供功能如下，詳細資料可以參考[定價方案](https://kktix.com/pricing)
- 同時間 5 場活動
- 單場 500 人數上限
- SSL 安全連線
- 自訂報名表單
- 電子票券
- KKTIX Manager App 驗票
{% endnote %}

- [活動通 Accupass](https://www.accupass.com)
{% note primary %}
免費活動限制與提供功能如下，詳細資料可以參考[免費方案](http://www.accupass.com/event/feature)
- 600人以下(含600人)
- 自訂報名表單
- 電子票券
- 快速驗票系統
{% endnote %}

- [BeClass](http://www.beclass.com/)
{% note danger %}
免費提供的平台，所以有廣告。
- 沒有人數限制
- 提供團體報名
- 每個帳號都提供三組自訂表格格式
- 提供嵌入語法
{% endnote %}

- [myRegie](http://www.myregie.tw/)
{% note warning %}
myRegie的收費方式是依照每一場活動來計費，免費活動限制與提供功能如下，詳細資料可以參考[免費/加值服務之費用及功能比較](http://www.myregie.tw/content.htm?name=pricing)
- 100人以下
- 報名安全機制(https)
- 提供嵌入語法
{% endnote %}

- [報名王](http://www.signupking.com.tw/)
{% note default %}
- 每個帳號提供3個活動額度
- 每個活動最多30個場次
- 不限報名人數
- 可下載報名資料
- 報名表最多可有30個欄位
{% endnote %}

- [eventsmart](https://eventsmart.com/)
{% note info %}
免費活動限制與提供功能眾多，詳細資料可以參考[免費方案](https://eventsmart.com/pricing/)
- 不限活動票卷數量
{% endnote %}

- [eventbrite](https://www.eventbrite.com/l/registration-online/)
{% note warning %}
按照售出票卷承收手續費，所以免費活動免費票卷是免費使用的，詳細功能資料可以參考[feature](https://www.eventbrite.com/features/)
{% endnote %}

- [Eventzilla.net](http://www.eventzilla.net/home)
{% note primary %}
Free for free events，所以免費活動免費使用的，詳細功能資料可以參考[feature](https://www.eventzilla.net/en/features)
{% endnote %}

- [ticketbud](https://ticketbud.com/)
{% note danger %}
Free for free events，所以免費活動免費使用的，詳細功能資料可以參考[feature](https://www.eventzilla.net/en/features)
{% endnote %}

- [123contactform](https://www.123contactform.com/)
{% note default %}
免費活動福利比較少，限制與提供功能如下，詳細功能資料可以參考[feature](https://www.123contactform.com/features-matrix.html)
- 5 個表單
- 每個表單限制 10 個項目欄位
- API 呼叫每天 100
{% endnote %}

## 表單系統
類似 google 問卷的，線上問卷與小型活動的好選擇。
- [Google 表單](https://www.google.com.tw/intl/zh-TW/forms/about/)
- [wufoo](https://www.wufoo.com/)
可以免費使用，另有[其他方案](https://www.wufoo.com/pricing/1/)選擇
- [pandaform](https://pandaform.com/)
這只有 14 天的免費使用，之後都是付費功能；[各項付費方案](https://pandaform.com/pricing-sign-up/)
