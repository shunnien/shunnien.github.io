Cmder命令列工具alias的注意設定

發表於 2017-05-24 | 分類於 程式技術筆記 | | 本文閱讀量 次
cmder 的自訂 alias 沒有作用

Introduction

前陣子因為 WannaCry 勒索病毒肆虐，一堆系統安全性漏洞開始更新，我的 win10 也使用還原更新到 1703，想說順便將一些免安裝軟體，重新調整一下，才發現 Cmder 的小設定。

順便提供一下大家查詢 win10 版本的方式，請使用 win+R 呼叫出執行視窗，然後執行 winver 即可。

Conetent

在 Cmder 中，我的預設執行命令列工具都是設定 CMD ，所以自訂 alias 在 powershell 中，應該是沒有喔。
解決方式也很簡單，直接啟用命令列工具。

滑鼠右鍵，選擇內容，然後開啟使用舊版主控台即可。
